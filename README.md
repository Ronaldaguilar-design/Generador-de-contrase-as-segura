# Generador Seguro de Contraseñas

## Descripción

Proyecto académico desarrollado en Python para generar contraseñas seguras personalizables.

## Funcionalidades

- Configurar longitud de la contraseña
- Incluir mayúsculas
- Incluir números
- Incluir símbolos
- Generar contraseña aleatoria
